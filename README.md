# daypop-search-box

## Usage
install
```
npm install daypop-search-box
```

```
<template>
    <SearchBox />
    <SearchBox lng="arabic" />
    <SearchBox lng="en" />
</template>
<script>
import SearchBox from 'daypop-search-box'
import 'daypop-search-box/dist/search-box.css'

export default {
    name: 'App',
    components: {
        SearchBox
    }
}
</script>
```
